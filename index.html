How to Play

Each column represents a range of alphabetically ordered items:

  • B → Starting with A, B, C, D, E
  • I → Starting with F, G, H, I, J
  • N → Starting with K, L, M, N, O
  • G → Starting with P, Q, R, S, T
  • O → Starting with U, V, W, X, Y, Z

Playing the Game

  1. When an item is called, check your bingo card.
  2. If you have it, click on it to mark it.
  3. Win the game by completing a line in any direction.

Game Buttons

Randomise. Generates a new bingo card.

Clear. Removes all marks from your current card.
